## Building

Create a virtual environment to install the needed filter:
```bash
    python -m venv venv
    source venv/bin/activate.<your shell>
    # Install the needed packages
    pip install pandoc-include pandoc-mermaid-filter
```

Build the PDF:

```bash
    make thesis-pdf
```
